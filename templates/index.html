Exchange setup guide
🔗 Sign in to Alpaca
  1. If you don't have an Alpaca account, click the link above to create one.
  2. Log into your Alpaca Dashboard.
  3. On the right-hand panel, locate the "API Keys" section.
  4. Click "Generate New Keys" to view your Key ID and Secret Key.
🔗 Sign in to OKX
  1. If you don't have an OKX account, click the link above to create one.
  2. Log into your account and hover over the Profile icon in the top right menu.
  3. Click on "API Keys" from the dropdown.
  4. Click "Create V5 API Key", choose an API Name, and set a custom Passphrase (keep note of this passphrase!).
  5. Set permissions to allow "Read" and "Trade", complete your security verification, and copy your API Key and Secret Key.
and then come back here and enter the keys to start trading.
